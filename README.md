# claudemods spitfire limine theme

## Options in the script

1) **Install limine** – Works from existing running GRUB systems
2) **Install theme** – Applies my Spitfire theme
3) **Remove theme and restore backup** – Reverts to the previous theme
4) **Edit limine.conf manually** – Opens the configuration for manual editing
5) **Exit** – Quits the script

---

## Installation

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/claudemods/spitfire-limine-theme/main/install/install-from-github.sh)"
```
